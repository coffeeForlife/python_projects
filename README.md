# Point Of Sale (POS) system example project 

## This project is a POS system written in python that can be run on the IDE terminal and is designed to simulate simple restaurant ordering process

This project was created using various Python fundamentals such as Functions, Loops, Classes and Data Structures

## Features of my POS:
* Create the Menu by creating categories and adding food or drink items to those categories
* Remove items from the Menu
* Select Tables and add orders to the individual tables
* View the reciept by choosing the table number
* Close the sale for a table, upon payment by the customer
* Generate the daily sales report


## Language and modules used: 
* Python 3
* This project only used builtin python modules:
    1. time
    2. datetime
